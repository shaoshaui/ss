# 画一个三角形

## 正三角形

```css
.box {
  width: 0;
  height: 0;
  border: 40px solid;
  border-color: transparent transparent red;
}
```

## 右直角三角形

```css
.box {
  width: 0;
  height: 0;
  border: 40px solid;
  border-color: transparent transparent red red;
}
```

## 左直角三角形

```css
.box {
  width: 0;
  height: 0;
  border: 40px solid;
  border-color: transparent red red transparent;
}
```
